
## SportTrack

This project shows is the showcase of the basic use of express_webapp.
### FilePath

```
README.md
|__
    controllers //php controllers
    data //.sql and .db
    express_webapp //core of the application
    img //background image
    model //php models
    sport-track-db //models
    static //home for the js gps functions
    view //web pages
config.php //php routes
index.php //php landing page
```


## Execute

#### PHP
To deploy this project using php run

```
cd SportTrack
php -S localhost:port
```

Then go to on your web browser

```
http://localhost:port/logincontroller
```

#### JS
To deploy this project using express webapp run

```
cd SportTrack/express_webapp
npm start
```

Then go to on your web browser

```
http://localhost:3000/login
```
